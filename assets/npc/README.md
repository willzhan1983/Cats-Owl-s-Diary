# NPC 图像资产说明

把正式 NPC PNG 放在这个目录后，游戏会自动优先显示图片；图片不存在时会继续使用 emoji 兜底，不会影响运行。

建议文件名：

- `lily.png` 兔兔 Lily
- `coco.png` 松鼠 Coco
- `nono.png` 刺猬 Nono
- `bibi.png` 小鸟 Bibi
- `dodo.png` 小鸭 Dodo
- `frogy.png` 青蛙 Frogy
- `swan.png` 天鹅 Swan
- `flora.png` 火烈鸟 Flora
- `toto.png` 老乌龟 Toto
- `mumu.png` 发光蘑菇 Mumu
- `buddy.png` 小狗 Buddy
- `sign.png` 路牌 Sign
- `bear.png` 咖啡馆老板 Bear
- `ruru.png` 浣熊 Ruru
- `fifi.png` 小鱼 Fifi
- `boat.png` 小木船 Boat
- `ollie.png` 猫头鹰 Ollie
- `star.png` 星星 Star

推荐规格：

- PNG 透明背景
- 512 x 512 px
- 圆润绘本风
- 与封面小猫、猫头鹰保持统一风格
